# 一起学习Linux

**本章内容分两部分来学Linux**

- [第一部分](./NOTE_BASE.md): Linux基础，主要掌握Linux的一些基本概念和命令，从文件系统、基本命令、软件包、用户和组管理、权限管理等方面开始。
- [第二部分](./NOTE_HIGH.md): Linux高级部分，主要掌握Linux的Shell编程，能够编写Shell脚本，使用Shell进行高级操作。

> 两部分内容会交替进行，先学一部分，再学一部分。
